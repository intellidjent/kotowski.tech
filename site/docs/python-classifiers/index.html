Python Classifiers

Starting SDK v10.0 you can extend your SDK configuration with a set of Python classifiers based on Scikit-learn, MXNet, etc. SDK extension allows using ML algorithms implemented in Python with pipelines.

Here's a list of supported classifiers:

  • GenericPythonClassifier
  • DpLogisticRegressionClassifier
  • DpSvmClassifier
  • AdaBoostClassifier
  • BaggingClassifier
  • DecisionTreeClassifier
  • ExtraTreeClassifier
  • ExtraTreesClassifier
  • GradientBoostingClassifier
  • KNeighborsClassifier
  • LinearSupportVectorClassifier
  • LogisticRegression
  • LogisticRegressionCV
  • RandomForestClassifier
  • StochasticGradientDescentClassifier

Setting up development environment

This section contains instructions for setting up the local development environment for proper building and running a custom model with Python model support.

Installing system packages

Install the required system packages (Python, libraries, compiler, etc.).

Windows
macOS
Linux


On Windows, follow these steps:

  1. Install Visual Studio 2017 Community Edition.

  2. Install Python 2.7.16 or higher.

  3. Check that pip is installed.

    <python_installed_dir>/python -m pip

If an error occurs, install pip:

  1. Download get-pip.py with curl or via browser.

    curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

  2. Install pip.

    <python_installed_dir>/python get-pip.py


On macOS, proceed with these steps:

  1. Download and install Python 2.7.16 (or higher).

  2. Install the Compiler (GNU Compiler Collection) with the following command:

    brew install gcc


On Linux, run one of the following commands:

  • Install Python on Fedora.

    dnf install gcc python2 python-devel libstdc++

  • Install Python on Ubuntu.

    apt-get install gcc python2.7 python-dev python-pip

Finally, add Python 2.7 <python_installed_dir> to the environment variable PATH, if it's not there already.

Installing Python packages

Before proceeding with package installation quit all instances of Terminal or Command line and start a new one.

To install the required Python packages, run the following command:

python -m pip install --user pip==19.0.2 setuptools==40.8.0 wheel==0.33.0 mxnet==1.3.1 scikit-learn==0.20.2 mock==2.0.0

Using virtual environment

If you want to use a virtual environment, follow these steps:

  1. Set the environment variables with this command:

    PYTHON_BIN=<path_to_python_binary_file>

  2. Use these steps to set the variables:

    • MacOS: ~/.bash_profile > export <VARIABLE>=<VALUE>
    • Linux: ~/.bashrc > export <VARIABLE>=<VALUE>
    • Windows: My Computer > Properties / Environment variables > <VARIABLE>=<VALUE>

  3. Quit and restart Terminal or IDE.

Using Python classifiers

To implement a Python classifier, use the Config @Named entity in your SDK configuration.

Here's an example of implementing RandomForestClassifier with default parameters.

@Named("Config")
public Classifier Config() {
    return RandomForestClassifier.builder()
            .build();
}

Here's an example implementing RandomForestClassifier with custom parameters, such as maxDepth, estimators, and maxFeatures.

@Named("Config")
public Classifier mlConfig() {
    return RandomForestClassifier.builder()
            .maxDepth(10)
            .estimators(100)
            .maxFeatures(MaxFeaturesType.LOG2)
            .build();
}
