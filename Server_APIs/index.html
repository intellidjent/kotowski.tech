Server APIs

Server APIs

API requests used to power game servers.

getFullInventory

Recursively get profile's and its children's entitlements and virtual currency balances.

Request URL

POST https://api-wgie.wargaming.net/server/api/v1/getFullInventory

Request Sample

{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "profile_id": 93503539
  }
}

Curl

curl -X POST --header 'Content-Type: application/json' --header 'Accept: application/json' --header 
'x-freya-server-api-key: test' -d '{
  "header": {
  "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "profile_id": 93503539
  }
}' 'https://api-wgie.wargaming.net/server/api/v1/getFullInventory'

Parameters

Name Description Type
x-freya-server-api-key Authentication API key. string
message_id For tracking or debugging purposes. Returns back in the response. string, optional
profile_id Profile ID returned during the authentication flow. int

Response Sample

{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "success": true,
    "result_message": "string",
    "result_code": "string",
    "profile": {
      "entitlements": [
        {
          "entitlement_code": "regular_repair_kit",
          "amount": 2,
          "expiration": 0
        }
      ],
      "currencies": [
        {
          "currency_id": "gold",
          "currency_code": "1",
          "amount": "1000"
        }
      ],
      "profile_id": 93503539,
      "game_data": "Game supplied string when child profile was created.",
      "children": [ {} ]
    }
  }
}

getFullInventoryFromSession

Batch operation combining validateClientSession and getFullInventory.

Request URL

POST
https://api-wgie.wargaming.net/server/api/v1/getFullInventoryFromSession

Request Sample

{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "client_session": "dXHGORzPvM7fAgcB4hHSkw",
    "fingerprint": "21ad5190-228f-440a-a338-ca84606f57c9"
  }
}

Curl

curl -X POST --header 'Content-Type: application/json' --header 'Accept: application/json' --header 
'x-freya-server-api-key: test' -d '{
  "header": {
    "message_id": "5865597c-9853-4d6c-815a-622be4b799be"
  },
  "body": {
    "client_session": "dXHGORzPvM7fAgcB4hHSkw",
    "fingerprint": "21ad5190-228f-440a-a338-ca84606f57c9"
  }
}' 'https://api-wgie.wargaming.net/server/api/v1/getFullInventoryFromSession'

Parameters

Name Description Type
x-freya-server-api-key Authentication API key. string
message_id For tracking or debugging purposes. Returns back in the response. string, optional
client_session Client session token. string
fingerprint A unique device fingerprint that stays consistent for a given user and device. string, optional

getPurchaseStatusByOrderId

Gets status of a purchase by Order Id.

Request URL

POST
https://api-wgie.wargaming.net/server/api/v1/getPurchaseStatusByOrderId

Request Sample

{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "order_id": "12334561233257",
    "profile_id": 93503539
  }
}

Curl

curl -X POST --header 'Content-Type: application/json' --header 'Accept: application/json' --header 
'x-freya-server-api-key: test' -d '{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "order_id": "12334561233257",
    "profile_id": 93503539
  }
}' 'https://api-wgie.wargaming.net/server/api/v1/getPurchaseStatusByOrderId'

Parameters

Name Description Type
x-freya-server-api-key Authentication API key. string
message_id For tracking or debugging purposes. Returns back in the response. string, optional
order_id Original order ID from the purchase. string
profile_id Profile ID returned during login. int

Response Sample

{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "success": true,
    "order_id": "string",
    "purchase_status": "COMMITTED"
  }
}
