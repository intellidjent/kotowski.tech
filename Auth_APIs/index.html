Platform API

Wargaming Network Platform provides a simple and consistent interface into the platform with patterns familiar to developers. API gateways serve as a single entry point into the system making the integration process easier, without any need to learn the underlying technologies.

Wargaming Network Platform can be accessed through the following API Gateways:

  • Authentication API
  • Server API
  • Client API (coming 2017)

Try these APIs in Swagger:

  • Auth API
  • Server API

Auth APIs

API requests used for player authentication.

loginWithAuthToken

This request is SERVER ONLY. Creates an authenticated user session from an Auth Token obtained from WGNI.

Request URL

POST https://api-wgie.wargaming.net/auth/api/v1/loginWithAuthToken

Request Sample

{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "region": "na",
    "auth_token": "JC0yZReLvfQPi76y2F5A4Q",
    "wgid": 18414499,
    "fingerprint": "a5d1cf90-f38c-4775-80cd-84e3fb76b509",
    "create_remember_me": true,
    "client_ip": "178.172.239.244",
    "auth_token_target_application": "xmppcs"
  }
}

Curl

curl -X POST --header 'Content-Type: application/json' --header 
'Accept: application/json' --header 
'x-freya-server-api-key: test' -d '{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "region": "na",
    "auth_token": "JC0yZReLvfQPi76y2F5A4Q",
    "wgid": 18414499,
    "fingerprint": "a5d1cf90-f38c-4775-80cd-84e3fb76b509",
    "create_remember_me": true,
    "client_ip": "178.172.239.244",
    "auth_token_target_application": "xmppcs"
  }
}' 'https://api-wgie.wargaming.net/auth/api/v1/loginWithAuthToken'

Parameters

Name Description Type
x-freya-server-api-key Authentication API key. string
message_id For tracking or debugging purposes. Returns back in the response. string, optional
region SPA region. string, optional
auth_token Wargaming Auth Token. string
wgid Wargaming Account ID. int
fingerprint A unique device fingerprint that stays consistent for a given user and device. string, optional
create_remember_me Boolean indicating if a Remember Me Token should be created. boolean, optional
client_ip Client IP address. boolean, optional
auth_token_target_application If not null, will create Auth Token for specified application. string, optional

Response Sample

{
  "header": {
    "message_id": "9fb0c554-132d-441c-b32c-1e13929852c0"
  },
  "body": {
    "success": true,
    "result_message": null,
    "result_code": null,
    "profile_id": 93503539,
    "nickname": "Player1990",
    "client_session": "salAzJiufFWCCT3rXiA79A",
    "client_gateway": null,
    "remember_me": "2:4758258918403912621:77218061794558285652020638946378270086",
    "auth_token": "65079312",
    "wgid": 18414499
    "first_login": false,
    "restrictions": [],
    "pending_order_id": "0"
  } }

While most of the response values are self-explanatory, some may need clarification.

  • client_session — Client session token.
  • first_login — Indicates whether this login instance is the first ever for this player.
  • restrictions — If not empty, provides the details of player's restrictions (for example, bans).
  • pending_order_id — Initial Inventory Order ID. 0 if order is complete or there is no initial inventory. If non-zero — use getPurchaseStatusByOrderId.

loginWithEmail

This request is SERVER ONLY. Creates an authenticated session for the user using email and password.

Request URL

POST https://api-wgie.wargaming.net/auth/api/v1/loginWithEmail

Request Sample

{
  "header": {
    "message_id": "a38a87f2-27f1-4962-9262-fa540051da1a"
  },
  "body": {
    "fingerprint": "a5d1cf90-f38c-4775-80cd-84e3fb76b509",
    "region": "ru",
    "email": "player@wargaming.net",
    "password": "Player1990",
    "create_remember_me": true,
    "client_ip": "178.172.239.244",
    "auth_token_target_application": "xmppcs"
  }
}

Curl

curl -X POST --header 'Content-Type: application/json' --header 'Accept: application/json' --header 
'x-freya-server-api-key: test' -d '{
  "header": {
    "message_id": "a38a87f2-27f1-4962-9262-fa540051da1a"
  },
  "body": {
    "fingerprint": "a5d1cf90-f38c-4775-80cd-84e3fb76b509",
    "region": "ru",
    "email": "player@wargaming.net,
    "password": "Player1990",
    "create_remember_me": true,
    "client_ip": "178.172.239.244",
    "auth_token_target_application": "xmppcs"
  }
}' 'https://api-wgie.wargaming.net/auth/api/v1/loginWithEmail'

Parameters

Name Description Type
x-freya-server-api-key Authentication API key. string
message_id For tracking/debugging purposes. Returns back in the response. string, optional
fingerprint A unique device fingerprint that stays consistent for a given user and device. string, optional
region SPA region. string, optional
email Wargaming account email address. string
password Wargaming account password. string
create_remember_me Indicates if a Remember Me Token should be created. boolean, optional
client_ip Client IP address. boolean, optional
auth_token_target_application If non-zero, creates an Auth Token for a specified application. string, optional

Response Sample
